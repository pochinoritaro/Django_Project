## 仮想環境構築
1. CUI上から任意のフォルダのカレントディレクトリで`python -m venv env`を実行
2. `./env/Scripts/activate.bat`を実行する。
3. `pip install -r installed_modulelist.txt`を実行
<br>

## 実行方法
1. 仮想環境上で
2. `python manage.py runserver`を実行
3. コンソール上に出たアドレスにアクセス
